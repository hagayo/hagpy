:::explanation title="The core idea"
Create small, explainable commits before working remotely.
:::

:::codeExample title="Working example" filename="example.py"
```python
git switch -c feature/user-api
git add src tests
git commit -m "Add user creation service"
```
:::

:::checklist title="Key points"
- Commit one coherent change
- Review git diff before committing
- Never commit secrets or virtual environments
:::
