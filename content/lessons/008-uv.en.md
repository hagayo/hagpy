:::explanation title="The core idea"
Manage reproducible environments, dependency groups and upgrades.
:::

:::codeExample title="Working example" filename="example.py"
```python
uv add fastapi
uv add --dev pytest ruff pylint pytest-cov
uv lock
uv sync --frozen
```
:::

:::checklist title="Key points"
- Commit pyproject.toml and uv.lock
- Separate development dependencies
- Use frozen sync in CI
:::
