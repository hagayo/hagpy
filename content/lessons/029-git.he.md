:::explanation title="הרעיון המרכזי"
יוצרים Commits קטנים וברורים לפני שעובדים מרחוק.
:::

:::codeExample title="דוגמה מעשית" filename="example.py"
```python
git switch -c feature/user-api
git add src tests
git commit -m "Add user creation service"
```
:::

:::checklist title="נקודות חשובות"
- בצעו Commit לשינוי מלוכד אחד
- בדקו git diff לפני Commit
- אל תשמרו סודות או סביבה וירטואלית
:::
