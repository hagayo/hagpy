:::explanation title="הרעיון המרכזי"
מוצאים הגדרות קצרות למונחי השפה, הכלים, הבדיקות וה-Backend שמופיעים ב-HagPy.
:::

:::codeExample title="דוגמה מעשית" filename="example.py"
```python
API       Application Programming Interface
CI        Continuous Integration
Fixture   Reusable pytest setup
Linting   Static code-quality analysis
Schema    A contract that describes data
```
:::

:::checklist title="נקודות חשובות"
- הגדרות מחברות מושגים לתפקיד המעשי שלהם
- המונחים ניתנים לחיפוש בשתי השפות
- דוגמאות מבדילות בין מושגים דומים
:::
