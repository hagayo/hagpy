:::explanation title="The core idea"
Model values clearly with names and built-in types.
:::

:::codeExample title="Working example" filename="example.py"
```python
name: str = "Maya"
age: int = 24
active: bool = True
score: float = 9.5
```
:::

:::checklist title="Key points"
- Names point to objects
- Types describe valid operations
- Avoid encoding types inside names
:::
