:::explanation title="The core idea"
Use JSON as a data format without confusing it with JavaScript objects or Python dictionaries.
:::

:::codeExample title="Working example" filename="example.py"
```python
{"title":"Learn Python","done":false}
```
:::

:::checklist title="Key points"
- JSON is text on the wire
- Keys and strings use double quotes
- Dates and UUIDs become strings
:::
