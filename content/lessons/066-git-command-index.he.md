:::explanation title="הרעיון המרכזי"
מתקדמים בביטחון מבדיקה ו-Commits אל Branches, Remotes והתאוששות.
:::

:::codeExample title="דוגמה מעשית" filename="example.py"
```python
git status
git diff
git switch -c feature/name
git add src tests
git commit -m "Add feature"
git push -u origin feature/name
```
:::

:::checklist title="נקודות חשובות"
- בודקים לפני שמשנים מצב
- מעדיפים פקודות התאוששות הפיכות
- מסבירים השפעה מקומית לעומת מרוחקת
:::
