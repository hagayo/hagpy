:::explanation title="Welcome to HagPy"
You may already have an idea for a website, a small tool that saves time, or a product that other people could use. Until now, that idea may have existed mainly in your head. In this learning path, you will turn it into working code that opens in a browser and can be shared with others.

You do not need any previous programming experience. We will begin by installing the tools, explain every new concept in plain language, and build your knowledge gradually. At every stage, you will understand what you are doing, why it is needed, and how to confirm that it worked.
:::

:::explanation title="Why Python?"
Python lets you write clear code and move quickly from an idea to a visible result. Instead of struggling with complicated syntax at the beginning, you can focus on the problem you want to solve and how your product should behave.

Python can power backend services, process data, automate tasks, connect to databases, and integrate artificial intelligence services. It is approachable for beginners and is also used to run real production systems.
:::

:::checklist title="Within one week, you will be able to publish your first product"
- Prepare an organized development environment on your computer
- Create a Python project and manage its dependencies with uv
- Write code that receives information, processes it, and returns a result
- Build the first version of your own website or product
- Save your code with Git and GitHub
- Publish the project on the internet
- Send a working link to another person
:::

:::explanation title="How will you learn?"
Every topic begins with a simple explanation of the idea and the reason it exists. You will then see an example, break it into understandable parts, and run it yourself.

You will do more than copy code. You will change it, observe what happens, and add a small capability of your own. Each lesson moves from understanding to action, and every action brings you closer to a working product.
:::

:::checklist title="From idea to publication in seven days"
- Day 1: Install the tools and prepare your workspace
- Day 2: Create an organized project and write your first Python code
- Day 3: Add data, decisions, and actions to the product
- Day 4: Build an interface that runs in the browser
- Day 5: Connect the different parts of the product
- Day 6: Test, fix, and improve the experience
- Day 7: Publish the product and share a working link
:::

:::explanation title="More than making code work"
Throughout the path, you will also learn how professional developers think: how to divide a problem into small parts, choose clear names, organize a project, and investigate an error without being intimidated by it.

Later, you will add automated tests, code-quality checks, an API built with FastAPI, a database, and an organized deployment process. Each tool will be introduced when you can clearly understand the problem it solves.
:::

:::exercise title="Your first task"
Write one sentence describing the website, tool, or product you would like to publish online. You do not need to know how to build it yet.

Save that sentence. You will return to it throughout the learning path, divide it into small capabilities, and watch a general idea gradually become a working product.
:::

:::explanation title="Ready to begin"
You do not need to remember everything in advance. You only need to begin, experiment, make mistakes, read the result, and continue to the next step.

In the next lesson, you will prepare the workspace where you will write, run, and publish your first project.
:::
