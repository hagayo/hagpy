:::explanation title="The core idea"
Install Git for Windows and configure the identity attached to commits.
:::

:::codeExample title="Working example" filename="example.py"
```python
git --version
git config --global user.name "Your Name"
git config --global user.email "you@example.com"
```
:::

:::checklist title="Key points"
- Download Git only from the official site
- Keep Git Credential Manager enabled
- Use the email verified by GitHub
:::
