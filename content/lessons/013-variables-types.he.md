:::explanation title="הרעיון המרכזי"
מייצגים ערכים באמצעות שמות וטיפוסים מובנים.
:::

:::codeExample title="דוגמה מעשית" filename="example.py"
```python
name: str = "Maya"
age: int = 24
active: bool = True
score: float = 9.5
```
:::

:::checklist title="נקודות חשובות"
- שמות מצביעים לאובייקטים
- טיפוסים מגדירים פעולות תקינות
- אל תקודדו טיפוסים בתוך שמות
:::
