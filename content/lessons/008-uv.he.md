:::explanation title="הרעיון המרכזי"
מנהלים סביבת פרויקט ניתנת לשחזור, ספריות, קבוצות תלויות ושדרוגים.
:::

:::codeExample title="דוגמה מעשית" filename="example.py"
```python
uv add fastapi
uv add --dev pytest ruff pylint pytest-cov
uv lock
uv sync --frozen
```
:::

:::checklist title="נקודות חשובות"
- שמרו את pyproject.toml ואת uv.lock ב-Git
- הפרידו תלויות פיתוח
- השתמשו ב-frozen sync ב-CI
:::
