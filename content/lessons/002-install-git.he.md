:::explanation title="הרעיון המרכזי"
מתקינים Git for Windows ומגדירים את הזהות שמוצמדת ל-Commits.
:::

:::codeExample title="דוגמה מעשית" filename="example.py"
```python
git --version
git config --global user.name "Your Name"
git config --global user.email "you@example.com"
```
:::

:::checklist title="נקודות חשובות"
- מורידים Git רק מהאתר הרשמי
- משאירים את Git Credential Manager פעיל
- משתמשים באימייל שאומת ב-GitHub
:::
