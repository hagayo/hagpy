:::explanation title="The core idea"
Move confidently from inspection and commits to branches, remotes and recovery.
:::

:::codeExample title="Working example" filename="example.py"
```python
git status
git diff
git switch -c feature/name
git add src tests
git commit -m "Add feature"
git push -u origin feature/name
```
:::

:::checklist title="Key points"
- Inspect before changing state
- Prefer reversible recovery commands
- Explain local versus remote effects
:::
