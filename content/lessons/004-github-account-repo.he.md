:::explanation title="הרעיון המרכזי"
יוצרים ומאמתים חשבון GitHub ולאחר מכן Repository ציבורי ריק.
:::

:::codeExample title="דוגמה מעשית" filename="example.py"
```python
Repository name: hagpy-first-site
Visibility: Public
README: unchecked
.gitignore: None
License: None
```
:::

:::checklist title="נקודות חשובות"
- מאמתים את כתובת האימייל
- יוצרים Repository ריק לפרויקט מקומי קיים
- מעתיקים את כתובת ה-HTTPS של ה-Repository
:::
